# Airfoil Documentation

Airfoil is an open source CLI tool made by Airship LLC to bootstrap, develop, and maintain React
Native apps.

[Check out the source code](https://github.com/teamairship/airfoil)

Many thanks to our friends at [Infinite Red](https://infinite.red/), the makers of
[Ignite](https://github.com/infinitered/ignite) and
[Gluegun](https://github.com/infinitered/gluegun). Airfoil is heavily inspired by Ignite, and is
built with Gluegun.
